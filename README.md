# PDF RAG Chatbot

A Retrieval-Augmented Generation chatbot built with:

- LangChain
- FAISS
- HuggingFace Embeddings
- Gemini API
- PyPDF

Features:

- Ask questions from PDFs
- Vector Search using FAISS
- Anti-hallucination prompt
- Terminal chat interface

Run:

pip install -r requirements.txt

python main.py
